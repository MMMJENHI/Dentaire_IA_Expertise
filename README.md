# 🦷 Dentaire IA : Analyse du Tiers Apical

Ce projet utilise l'Intelligence Artificielle et le traitement d'image pour assister les dentistes dans le diagnostic des pathologies apicales (Dent 16).

### 🚀 Fonctionnalités
- **Variable H** : Mesure de la densité relative du matériau d'obturation.
- **Focus Tiers Apical** : Isolation automatique des derniers 33% de la racine.
- **Expert IA** : Diagnostic automatique (Sain / Fuite / Pathologie).

### 📊 Seuils de Référence
- **H > 0.90** : Étanchéité conforme.
- **H < 0.45** : Suspicion de réaction apicale (lésion).

### 🛠️ Installation
1. `pip install -r requirements.txt`
2. `streamlit run dentaire_ia.py`
